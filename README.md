# CRUD Java con MySQL

Proyecto realizado en Java utilizando Programación Orientada a Objetos.
Permite crear, leer, actualizar y eliminar registros en una base de datos MySQL.

## Tecnologías
- Java
- MySQL
- JDBC
- NetBeans

## Funcionalidades
- Alta de registros
- Listado
- Modificación
- Eliminación
